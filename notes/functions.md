# Funciones

## Declaración

- No devuelve ningún valor

```py
def nombre_funcion():
	# Cuerpo de la función
	# Bloque de instrucciones
```

- Devuelve un valor

```py
def nombre_funcion():
	# ...
	return valor_devuelto
```

- Con parámetros

```py
def nombre_funcion( parametro_1, parametro_2, etc ):
	# ...
	# return valor_devuelto
```

## Llamada

- Sin parámetros

```py
nombre_funcion()
```

- Con parámetros

```py
nombre_funcion( valor_1, valor_2, etc)
```
