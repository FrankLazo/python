# Instrucción `pass`

- Se usa para rellenar un bloque de código que aún no se desea escribir.
- Así el programa se puede ejecutar sin que salte error en la consola.

```py
if expresion_condicional:
	pass
```

```py
for i in range(n):
	pass
```

```py
while expresion_condicional:
	pass
```

```py
def funcion_x():
	pass

funcion_x()
```
